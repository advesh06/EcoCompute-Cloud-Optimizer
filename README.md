# EcoCompute Cloud Optimization Suite 🌱💰

An end-to-end full-stack cloud infrastructure monitoring and sustainability orchestration platform.

## Features
- **Landing Dashboard:** Ingest infrastructure node metrics dynamically via template mock sets.
- **Telemetry Detection Sandbox:** Live custom calculation sandbox evaluating real-time server metrics against regional grid factors.
- **Resource Optimization Matrix:** Policy engine mapping utilization profiles directly to absolute savings rates.
- **Executive Analytics Hub:** Multi-region charts and an automated AI Insights engine calculating carbon and monetary deltas.

## Tech Stack
- **Frontend:** React.js, Recharts, Axios, CSS Flexbox Dark Theme
- **Backend:** Node.js, Express.js, Mongoose
- **Database:** MongoDB
